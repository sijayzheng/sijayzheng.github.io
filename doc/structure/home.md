## 数据结构

| 数据结构 | 特点           | 实现方式         |
|------|--------------|--------------|
| 数组   | 内存中连续、长度固定   |              |
| 链表   | 内存中不必连续      |              |
| 列表   | 元素有序集合       | 动态数组         |
| 栈    | FILO         | 链表或数组        |
| 队    | FIFO         | 链表或数组        |
| 双向队列 | 两端都可入队出队     | 链表或数组        |
| 哈希表  | 高效           | 数组加链表或数组加红黑树 |
| 树    | 层级结构、体现派生关系  | 链表或数组        |
| 堆    | 满足特定条件的完全二叉树 |              |
| 图    | 自由度高         | 链表或数组        |
